一、右侧面板

1、初始化了雷达信息的显示方式
	
	调整初始图像以彩色、2.4海里、阈值电压1.0V方式显示。

2、添加了图像放大、连放功能

	图像放大显示分为0.75海里、1.5海里、2.4海里三个量程，分别对应600点中的

   200点、400点和600点显示。
	
	连放方式分32幅连放和时间序列连放：32幅连放连续显示当前序列的32幅 
  
   文件，若当前序列不足32幅，则显示到该序列最后一幅并弹出提示对话框；时间序列连    
   放则按文件创建次序查找并显示出每序列的第一副文件，直到该文件夹内文件查找结束
   
   为止。在连放过程中，设有暂停和继续功能。

3、添加了伪彩色显示方式

	在伪彩方式显示时，图象左侧弹出伪彩色标，色标值随阈值动态变化。

4、在右侧雷达信息显示框中添加显示了船首信号的触发线数和方位线数	

二、左侧图像窗口

1、画当前图像前以红色字体在四角显示当前图像的文件名称和相关导航信息。

2、以箭头方式标出了当前图像的风向信息。

3、雷达图像相邻距离环间隔750米，六等分图像半径。

三、导航信息窗口

点击查看->导航信息，弹出导航信息窗口

	左侧导航信息1和2的有效标识位当相关导航信息无效时，以红色字体显示。	

	右侧显示该幅图像导航信息1的变化曲线，横坐标为导航信息1的组数，纵坐标分

   别为艏向角、纵摇角、横摇角在所有组里的边界值。

四、右键弹出对话框

	以列表方式显示了该幅图像各方位信号中的触发线数和总触发线数。

五、修改了软件中因一些不正确操作所引起的读取内存错误、循环、死机现象。
